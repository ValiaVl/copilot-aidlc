# User Personas — Team Knowledge Base

## Persona: Team Member

| Attribute | Description |
|-----------|-------------|
| **Name** | Team Member |
| **Role** | Any member of the team using the knowledge base |
| **Goals** | Document recurring team knowledge, find answers quickly, keep information up to date |
| **Frustrations** | Answering the same questions repeatedly, outdated docs that mislead, can't find the right article |
| **Tech Comfort** | Comfortable with markdown and web apps |
| **Usage Patterns** | Creates articles when a question comes up repeatedly; searches before asking; edits others' articles to improve accuracy; flags stale content when noticing outdated information |
| **Access Level** | Full access — can create, read, edit, search, tag, and flag any article |

### Key Motivations
- **Save time** by writing it down once instead of explaining repeatedly
- **Find answers fast** through search and tag filtering
- **Trust the content** — stale flagging helps identify unreliable articles
- **Collaborate openly** — anyone can improve any article
